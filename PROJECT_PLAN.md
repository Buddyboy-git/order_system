# Project Plan: Order System

## Goals
- Centralize vendor data extraction, transformation, and loading.
- Provide a robust, self-verifying AJAX search UI for products.
- Support both local and production environments with easy config switching.
- Enable modular, maintainable code for future features.

## Completed Features (Backfilled)
- Local/production config override for PHP scripts
- AJAX backend API for product search
- Modular JavaScript search UI
- Robust error handling in HTML/JS
- Data import scripts (CSV, SQL)
- AI product info modal (with Wikipedia/Unsplash fallback)
- Documentation and deployment guides

## Workflow
- Edit and version-control everything in this repo.
- Sync PHP files to XAMPP as needed for local testing.
- Use `/docs/` for outlines, specs, and deployment instructions.
